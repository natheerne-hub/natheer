# Medical Data Analysis for Diabetes 🩺📊

هذا المشروع مخصص لتحليل واستكشاف وتنظيف بيانات مرضى السكري لبنائها وتجهيزها بشكل صحيح قبل تمريرها لنماذج التعلم الآلي.

## 📁 محتويات المستودع (Repository Contents):
- `diabetes.csv`: ملف البيانات الطبي الأصلي.
- `Untitled0.ipynb`: دفتر ملاحظات جوبيتر (Jupyter Notebook) الذي يحتوي على أكواد بايثون والرسومات البيانية.

## 🛠️ الخطوات التي تمت في المشروع (Key Pipeline):
1. **استكشاف البيانات (Data Exploration):** معرفة أبعاد الجدول والتحقق من أنواع البيانات وغياب القيم الفارغة التقليدية.
2. **معالجة البيانات المفقودة الصامتة (Handling Implicit Missing Data):**
   - تم اكتشاف قيم صفرية غير منطقية طبياً في أعمدة (الجلوكوز، ضغط الدم، سماكة الجلد، الإنسولين، ومؤشر كتلة الجسم).
   - تم استبدال جميع هذه الأصفار بقيم فارغة `NaN`.
3. **التعويض الذكي (Imputation):** تعويض القيم الفارغة باستخدام **الوسيط الإحصائي (Median)** لكل عمود لتفادي تأثير القيم المتطرفة.
4. **تحليل التوزيعات والقيم الشاذة (Outliers Analysis):** رسم المخططات الصندوقية (Box Plots) لمعرفة توزيع البيانات ورصد الحالات الطبية المتطرفة (مثل الارتفاع الحاد في الإنسولين).

## 🧰 المكتبات المستخدمة (Libraries Used):
- Pandas & NumPy (لإدارة وتنظيف البيانات)
- Matplotlib & Seaborn (لرسم المخططات البيانية)
